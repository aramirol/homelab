# graylog_vagrant

Vagrantfile + shellscript to deploy Graylog appliance as VM in Oracle Virtualbox. 

* **How_To_Run**: Just play it -- vagrant up

## Info

* **Image_ID: bento/centos7**
* **var_mem: 4096**
* **var_cpu: 4**
* **ver_graylog: 3.2**
* **provider: Virtualbox** 

At the end of deployment u'll see the URL access, user & pass.

Have fun.
